Insert BEFORE `<div style={{ padding: '16px' }}>` and AFTER the opening `<div className="card-animate" style={cardStyle}>`:

```tsx
{dest.photos && dest.photos[0] && !imgError ? (
  <img
    src={dest.photos[0]}
    alt={dest.name}
    loading="lazy"
    onError={() => setImgError(true)}
    style={{
      width: '100%',
      aspectRatio: '3 / 2',
      objectFit: 'cover',
      display: 'block',
    }}
  />
) : (
  <div
    className="topo-bg"
    style={{
      width: '100%',
      aspectRatio: '3 / 2',
    }}
  />
)}
```

Key details per D-05, D-06, D-09:
- Full-width banner at the very top of the card (D-05)
- `aspectRatio: '3 / 2'` used on both the img and the fallback div so the slot height is identical (D-06, D-09)
- `objectFit: 'cover'` on the img so it fills without stretching (D-06)
- No explicit `height` set -- aspect-ratio controls the height (D-06)
- `loading="lazy"` added per Claude's Discretion for performance
- `display: 'block'` eliminates inline image gap
- The `onError` handler sets `imgError` state to `true`, triggering a re-render that shows the `.topo-bg` placeholder (D-09)
- The `.topo-bg` class uses the existing topographic SVG pattern from `src/index.css` (D-09)
- If `dest.photos` is undefined/empty OR `imgError` is true, the topo-bg placeholder shows at the same aspect ratio
