## Bugs fixed beyond plan scope

**Map.tsx creator UID bug** (root cause of `isCreator` always being false):
- `currentUser` was set to `membersWithLocation[0]` (first user sorted by UID) — not the logged-in user
- Fixed: new `useEffect` loads current user's profile by `currentUid` from Firebase Auth
- Impact: `creatorUid` in new trips now correctly reflects the logged-in user; "Open voting" button now visible

**Fuel price API swap** (fuelprice.io → Servo Saver):
- fuelprice.io rejected: paid ($49.50/yr) and Cloudflare blocked all server-side requests
- Implemented Servo Saver (Service Victoria) API: free, government-maintained, working
- Proxy architecture: Vercel Edge Function (`api/fuel-prices.ts`) + Vite dev middleware
- Averages U91 (petrol) and DSL (diesel) across all VIC stations

**TripDetail datetime input styling**:
- Replaced generic `datetime-local` with branded input: label, DM Sans, surface bg, rounded border

**padding/paddingBottom shorthand conflict** in TripSheet sticky footer resolved.
